# Star Linker Privacy Policy

This repository contains the privacy policy for the Star Linker mobile application.

## Overview

Star Linker is a puzzle game that respects user privacy. This privacy policy page explains how the app handles user data and privacy.

## Key Features

- **No Data Collection**: Star Linker does not collect personal information
- **Local Storage Only**: All game data is stored locally on the user's device
- **No Third-Party Sharing**: No data is shared with external parties
- **Child-Friendly**: Complies with privacy regulations for users under 13
