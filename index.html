Privacy Policy

1. Purpose of Personal Information Processing

Star Linker (the "App") processes personal information for the following purposes and does not use it for any other purposes beyond those listed below.

2. Personal Information Processing and Retention Period

Star Linker does not collect personal information, and all game data is stored only on the user's device.

Star Linker does not provide personal information to third parties.

4. Consignment of Personal Information Processing

Star Linker does not consign personal information processing tasks to third parties.

Users can delete the app at any time to remove all local data.

Star Linker implements the following security measures:
